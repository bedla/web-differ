Application for monitoring changes on specified web pages

As a user of this application you can create web-page monitoring to see changes in time on URL using specified CSS selector.

Application is public beta and you will need activation code to access all functionality.

You can login (create user account) using your Google Account.

You are not logged in, please go to login page to login using your Google account.

You will be requested to consent security permissions for this application.

  • One group of permissions is used for your identification (name, email, avatar) by this application.
  • Second permission is used to send you notification email using GMail.

When application sends you notification email it will looks like that you have send email to yourself.

Application mainly saves basic information about you (you can see them at user's profile page) and OAuth2 access and refresh tokens send to application by Google authentication process during login. These tokens are used to comunicate with Google API servers (sending notification email)
